Project "Lab 4 (train v1)"
------------------------------------------------------------------------
This project consists of a simple Java simulation of a train system, including cars and seats. The program is organized into multiple classes to model the behavior of a train and its components.
Author: Amina Hajiyeva
------------------------------------------------------------------------

Classes Overview
  Car: Represents a train car, which contains multiple seats.
  CarTest: Tests the functionality of the Car class.
  Seat: Models a seat within a car, which can be reserved or unreserved.
  SeatTest: Tests the functionality of the Seat class.
  Train: Combines multiple cars to simulate a complete train system.

Features
- Manage train cars and their seats.
- Reserve and release seats.
- Track seat availability.
- Perform unit testing to ensure correct functionality.
